## 项目构思
文章生成/续写功能

*外置一神经网络用作评分系统，记录用户喜好，每次用户需求时让大模型输出多篇备选文章

*添加一些用户可选的风格/情感tag、用户对文章的构思（e.g.情节如何发展）

*尝试左右大模型的输出倾向

*针对用户润色功能

**记录用户对文章的修改内容
